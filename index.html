Research & Technical Projects

Group Supervised Learning: Extending Self-Supervised Learning to Multi-Device Settings
Workshop paper, ICML 21
Nokia Bell labs, UK 2021
  • Proposed a novel problem Time-synchronous multi-device system (TSMDS) that is prevalent in the real world but hasn't been unexplored
  • Introduced Group supervised learning (GSL), a novel framework solving TSMDS by extending contrastive learning to multi-device setup increasing the complexity manifold
  • Surpassed the Fully-supervised and existing Self-supervised baseline by atleast 0.15 F1-score utilizing a fraction of labeled data

Link to the poster

RFID Tattoo: A wireless speech recognition platform
[Best Wearables Paper award at UbiComp 2020
(1 out of 121)]Carnegie Mellon University, ECE Summer 2019
  • Speech recognition platform for voice impairments through wafer-thin, battery-free and stretchable RFID Tattoos
  • Collected own sensor data and implemented Random Forest model calibrated on the stretch of tags to achieve state of the art 86% accuracy on a vocabulary size of 100 most common English words

Link to the accepted paper

Automated E-commerce Question-Answering system
               Flipkart, Summer 2020          

  • Generated synthetic queries from a limited set of user query to increase the dataset size by more than 30%
  • Combined the BERT and GPT-2 models for developing a target product-type classification system which would then prompt the text-generation model to answer user query in natural language all in real-time
  • System might be implemented to the company’s website, subject to quality check approval

Increasing the range of NFC communication Carnegie Mellon University, ECE Summer 2019

  • Conducted experiments to determine the effects of antenna design and its placement on NFC range and its tag response
  • Developed codes for software-defined radios USRP for skimming live NFC signal request-response as well as synchronously transmitting phase-shifted NFC signal from multiple USRPs to perform in-air beamforming

Deep Neural Matching Models for Graph Retrieval from Product Graph
InfoLab, Indian Institute of Technology Bombay, Autumn 2020

  • Developing a novel DNN approach for searching a subgraph from a corpus of sample graphs by scoring random walks in their product graph
  • Working on decreasing the query time to sub-linear time by employing hashing of corpus graphs

Answer-type tagging through constant embeddings
InfoLab, Indian Institute of Technology Bombay, Spring 2020

  • Improved the performance of an existing Answer-Type Inference (ATI) RNN based manuscript by almost 5% through conducting experiments in 10 different settings across 4 QA datasets
  • Outperformed the previous RNN system by 20% across the 4 QA datasets by deploying BERT architecture in ATI task

Abstractive Text Summarisation Indian Institute of Technology Bombay, Autumn 2019

  • Implemented an extractive text summarizer using the page rank algorithm, achieved a ROGUE score of f-32%. Latent Semantic analysis of the generated summary and the ground truth revealed an overlap of 79%
  • Comparatively, the abstractive summarizer built using seq2seq model with attention mechanism, achieved a ROUGE scores of only f-13%

Adversial Reprogramming of Neural Networks Indian Institute of Technology Bombay, Autumn 2019

  • Reprogrammed the six ImageNet classification models on MNIST and CIFAR-10 dataset to perform a task chosen by the attacker by finding a single adversarial perturbation added to all test inputs
  • Illustrated the vulnerability in neural networks that perform a task chosen by the adversary despite being not trained to do this task originally

Link to original work

Quarter to Sixth SenseInstitute Technical Summer Project, IIT Bombay, Summer 2018

  • Developed a quarter of the features of 'SixthSense', a wearable gestural interface that augments the physical world around us with digital information and lets us use natural hand gestures to interact with that information

Wireless Gaming Console Institute Technical Summer Project, IIT Bombay, Summer 2018

  • Made a gesture-controlled Wireless Game Console that worked on Raspberry Pi and uses Image Processing libraries to move the mouse cursor and control the arrow keys for the movement by detecting the relative change in the position and orientation of the console

Nuclear Survey
(Paper accepted at DAE-BRNS Symposium on Nuclear Physics, 2019) Tata Institute of Fundamental Research, Summer 2018

  • Created robust scripts for parsing nuclear values like energy levels, spin and half-life from the international atomic nuclear dataset that helped in standardizing the nuclear levels of a vast number of known nucleus

Link to the survey website

Novel Alternative to analyze MCQ questions Tata Institute of Fundamental Research, Summer 2018

  • Devised an alternate approach for educational assessment of the MCQ questions using Discriminant Index (DI) by arguing against the existing approach through comparative analysis on conjecture data and data from an international competitive examination

Link to the submitted paper

Secure Personal Cloud Indian Institute of Technology Bombay, Autumn 2018

  • Implemented a cloud based file storing system for Linux system with block level customizable file encryption schema allowing multiple clients to decrypt their data at real time simultaneously or share the same
  • Developed a bash interface along with a web client for an interactive user experience

Twitter Simulation Indian Institute of Technology Bombay, Spring 2018

  • Coded a basic version of twitter handle that you generally see in Racket language (Functional Programming Paradigm). It includes Post, Replies to Post, Search, Notification bar, Following and Follower features
  • Users can put posts on their timeline and even reply to other users posts similar to any social networking site
